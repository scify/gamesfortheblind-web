Message for screen reader users

Welcome, If you are using a screen reader we recommend switching to "Full Access Mode". This mode is designed to help different types of navigation:

  • Each page is divided into sections and each section is described by a title (headings navigation).
  • The most important sections are assigned to a role (landmark navigation).
  • On the top of each page you will find the quick links menu (internal link navigation). Each link is assigned to an accesskey (keyboard shortcut).

LEAP games for the blind

Installation Guide

Read the instruction below….

Installing the games:

Click on the icon of the game you want to download. You will then be transfered to another website, where there is a list with all the available games editions for the according operating system:

  • For windows, you choose between the files ending in win64_built.zip και win_built.zip. If you are not sure whether you use 32 or 64 bit, you try to run both executable files.

  • For mac, you choose the file ending in mac_built.zip.

  • For linux, you choose the file ending in _lin_built.zip.

2. The files are compressed (.zip.). Using the proper program in your computer, you decompress. After decompressing, you can find the game in the created file.

3. You double-click and start to play!
